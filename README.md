# KeySprint ⚡

**KeySprint** is a fast, minimalist typing speed trainer built with vanilla JavaScript, HTML, and CSS.
It helps users practice typing while tracking speed, accuracy, and consistency in real time.
1
---

## Features ✨

### Typing Test Modes

* **Word Mode** – practice with 10, 25, 50, or 100 words
* **Time Mode** – test your speed for 15s, 30s, 60s, or 120s
* Real-time word highlighting and character validation
* Smooth caret animation
* Instant visual feedback for correct and incorrect characters

---

### Statistics & Analytics

KeySprint provides detailed typing analytics:

* **WPM (Words Per Minute)**
* **Accuracy Percentage**
* **Raw WPM** (including errors)
* **Consistency Score**
* Character breakdown:

  * Correct characters
  * Incorrect characters
  * Extra characters
  * Missed characters

Live statistics update while typing so you can monitor your performance in real time.

---

### Customization

KeySprint includes several customization options:

**Themes**

* Dark
* Light
* Serika
* Dracula

**Typing Options**

* Toggle punctuation
* Toggle numbers
* Sound effects

Themes are saved automatically using browser storage.

### User Experience

KeySprint focuses on a clean and distraction-free typing environment:

* Minimal interface
* Smooth animations
* Blur effect when typing area is unfocused
* Keyboard shortcuts for quick actions
* Responsive layout for desktop and mobile

## Installation 🚀

### Prerequisites

* Node.js (v14 or higher)
* npm

### Setup

1. Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/keysprint.git
cd keysprint
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm start
```

4. Open in your browser

```
http://localhost:3000
```

---

## Usage 📝

1. Select **Word Mode** or **Time Mode**
2. Configure settings if needed
3. Click the typing area and start typing
4. View detailed results after the test
5. Restart anytime with the restart button or shortcut

---

## Keyboard Shortcuts ⌨️

| Shortcut    | Action              |
| ----------- | ------------------- |
| Space       | Move to next word   |
| Shift + Tab | Restart test        |
| Escape      | Unfocus typing area |
